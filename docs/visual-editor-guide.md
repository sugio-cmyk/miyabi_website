# ビジュアルエディタ操作手順書
## VK Blocks を使ったホームページ編集ガイド

---

## 📋 概要

このガイドでは、コードエディタで基本構造を貼り付けた後、ビジュアルエディタでVK Blocksを追加する手順を説明します。

---

## 🔧 事前準備

1. `home.html` をコードエディタに貼り付け済み
2. 「ビジュアルエディター」に切り替え済み
3. VK Blocks プラグインが有効化されている

---

## 📌 作業1: Trust Badges のアイコン追加

### 場所
「国家資格者が対応」「初回相談無料」「オンライン対応可」「中小企業専門」の上

### 手順
1. `[VK ICON: ...]` と書かれた段落を選択
2. ツールバーの「⋮」→「ブロックを削除」
3. 同じ位置で「＋」をクリック
4. 「アイコン」と検索 → 「**アイコン**」ブロックを追加
5. アイコン設定パネルで以下を設定：

| 項目 | 値 |
|------|-----|
| アイコン 1 | `fa-user-tie` (検索: user tie) |
| アイコン 2 | `fa-comments` (検索: comments) |
| アイコン 3 | `fa-laptop` (検索: laptop) |
| アイコン 4 | `fa-building` (検索: building) |
| サイズ | 48px |
| 色 | #1a365d (ネイビー) |
| 配置 | 中央 |

## 📌 作業2: 選ばれる理由 のアイコン追加

### 場所
「寄り添うサポート」「地域密着」「オンライン対応」カード内

### 手順
各カード内で同様に：
1. wp:html アイコン部分を削除
2. 「アイコン」ブロックを追加
3. 設定：

| カード | アイコン | サイズ | 色 |
|--------|----------|--------|-----|
| 寄り添うサポート | `fa-handshake` | 56px | #5a7a54 |
| 地域密着 | `fa-map-marker-alt` | 56px | #5a7a54 |
| オンライン対応 | `fa-video` | 56px | #5a7a54 |

## 📌 作業3: FAQ セクションの修正

### 現状
VK Blocks FAQ2 形式で書かれているが、エラーが発生

### 手順
1. FAQセクション全体（4つのFAQ）を選択
2. 「ブロックを削除」で全て削除
3. 同じ位置で「＋」→「新FAQ」と検索
4. 「**新FAQ**」ブロックを4回追加

### FAQ内容

| Q | 質問 |
|---|------|
| 1 | 相談は無料ですか？ |
| 2 | どのような企業が対象ですか？ |
| 3 | オンラインでの相談は可能ですか？ |
| 4 | 契約前に見積もりはもらえますか？ |

| A | 回答 |
|---|------|
| 1 | はい、初回のご相談は無料です。お気軽にお問い合わせください。ご相談内容をお伺いした上で、最適なサポート内容とお見積もりをご提案いたします。 |
| 2 | 香川県内の中小企業様を中心にサポートしております。業種は問いません。社会保険・労働保険の手続きや給与計算でお困りの企業様はお気軽にご相談ください。 |
| 3 | はい、Zoomなどのビデオ通話を使ったオンライン相談に対応しております。ご来所いただく必要がないため、お忙しい経営者様にもご好評いただいております。 |
| 4 | はい、ご契約前に必ずお見積もりをお出しします。ご依頼内容や従業員数などをヒアリングの上、明確な料金をご提示いたしますのでご安心ください。 |

## 💡 ヒント

### アイコンの検索方法
1. アイコンブロックを追加
2. 右サイドバーの「アイコン」設定パネル
3. 「アイコンを選択」をクリック
4. 検索ボックスにアイコン名を入力（英語）

### 色の設定
- ネイビー（プライマリ）: `#1a365d`
- オリーブグリーン（セカンダリ）: `#5a7a54`
- ゴールド（アクセント）: `#c9a227`

### 保存前の確認
- 「プレビュー」ボタンで表示確認
- モバイル表示も確認（レスポンシブ）
