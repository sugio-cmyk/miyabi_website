## 3. ターゲット

### 3.1 基本属性

| 項目 | 内容 |
|------|------|
| 対象 | 中小企業（従業員1〜50名程度） |
| 地域 | 香川県（坂出市中心、高松市、丸亀市） |
| 業種 | 全業種対応（製造業、小売業、サービス業など） |
| 意思決定者 | 経営者、人事・経理担当者 |

### 3.2 ペルソナ

#### ペルソナA：経営者タイプ
| 項目 | 内容 |
|------|------|
| 属性 | 50代男性、製造業、従業員20名 |
| 課題 | 労務管理を経理担当に任せているが法改正対応が不安 |
| ニーズ | 信頼できる専門家に丸投げしたい |
| 行動 | 「社労士 香川」で検索、知人の紹介 |

#### ペルソナB：担当者タイプ
| 項目 | 内容 |
|------|------|
| 属性 | 40代女性、小売業、経理兼務 |
| 課題 | 給与計算に時間がかかり本業に集中できない |
| ニーズ | 給与計算だけでも外注したいが費用が心配 |
| 行動 | 「給与計算 代行 香川」で検索、料金比較 |

### 3.3 ターゲットの課題

| 課題 | 解決策（当事務所の価値） |
|------|--------------------------|
| 労務手続きの負担 | 手続き代行で経営に集中 |
| 法改正への対応不安 | 専門家が最新情報を提供 |
| 人材確保・定着 | 就業規則整備で働きやすい職場づくり |
| 専門家への敷居の高さ | 気軽に相談できる雰囲気 |

## 5. サービス構成

### 5.1 3つのサービスカテゴリ

| # | サービス名 | 概要 | 詳細 |
|:-:|-----------|------|------|
| 1 | 社会保険・労働保険手続き | 届出代行 | 入退社手続き、算定基礎届、年度更新 |
| 2 | 給与計算代行 | 経理負担軽減 | 月例給与、賞与、明細作成 |
| 3 | 人事労務支援 | 経営サポート | 就業規則、助成金、労務相談 |

### 5.2 差別化ポイント

| 強み | 内容 | 競合との違い |
|------|------|--------------|
| 寄り添うサポート | 気軽に相談できる雰囲気 | 敷居の高さを感じさせない |
| オンライン対応 | Zoom等での相談も可能 | 来所不要で時間節約 |
| 地域密着 | 香川県内の中小企業に特化 | 地域事情を理解 |
| 丁寧な説明 | 専門用語を使わず分かりやすく | 経営者目線での提案 |

### 5.3 料金目安（税別）

| サービス | 料金 | 備考 |
|----------|------|------|
| 顧問＋給与セット | 月額2万円〜 | 50名で7.5万円程度 |
| 就業規則作成 | 10万円〜 | 規模により変動 |
| 給与計算のみ | 対応なし | 顧問とセット |
| 年末調整 | 対応なし | 提携税理士紹介 |
