## ログイン方法

1. ブラウザで `https://あなたのドメイン/wp-admin/` にアクセス
2. ユーザー名とパスワードを入力
3. 「ログイン」をクリック

> 💡 ブックマークしておくと便利です

## ブログ記事の投稿

### 新規投稿

1. 管理画面左メニュー → 投稿 → 新規追加
2. タイトルを入力
3. 本文を入力（ブロックエディタ使用）
4. 右側パネルでカテゴリを選択:
   - **法改正情報**: 法律改正の解説記事
   - **お役立ち情報**: お客様向けの実用的な情報
5. アイキャッチ画像を設定（推奨）
6. 「公開」をクリック

### よく使うブロック

| ブロック | 用途 |
|----------|-----|
| 段落 | 通常の文章 |
| 見出し | セクションタイトル（H2, H3を使用） |
| リスト | 箇条書き・番号付きリスト |
| 画像 | 写真・図表 |
| ボタン | お問い合わせへのリンクなど |

### 記事作成のコツ

- **タイトル**: 検索されやすいキーワードを含める
- **見出し**: H2 → H3 の順で使う（H1は使わない）
- **段落**: 1段落は3〜4行程度に
- **最後に**: お問い合わせへの導線を設置

## 固定ページの編集

### 編集方法

1. 管理画面左メニュー → 固定ページ → 固定ページ一覧
2. 編集したいページの「編集」をクリック
3. 内容を修正
4. 「更新」をクリック

### 事務所情報の更新

事務所概要ページで以下を更新できます:
- 所在地
- 電話番号
- 営業時間
- 代表挨拶

### Slack通知

お問い合わせがあると、設定したSlackチャンネルに通知が届きます。

### WordPress管理画面での確認

2. 受信したメッセージ一覧が表示されます

## バックアップ

### 自動バックアップ

UpdraftPlusにより週1回自動でバックアップされます。

### 手動バックアップ

大きな変更前には手動バックアップを推奨:

1. 設定 → UpdraftPlus バックアップ
2. 「今すぐバックアップ」をクリック
3. データベースとファイル両方を選択
4. 「今すぐバックアップ」

### 復元方法

1. 設定 → UpdraftPlus バックアップ
2. 「既存のバックアップ」から復元したい日付を選択
3. 「復元」をクリック
4. 復元する項目を選択 → 「復元」

## 更新作業

### 月1回の定期作業

1. **WordPress本体の更新**
   - ダッシュボード → 更新
   - 「今すぐ更新」（メジャーアップデートは様子を見る）

2. **プラグインの更新**
   - ダッシュボード → 更新
   - 更新可能なプラグインを選択 → 「プラグインを更新」

3. **テーマの更新**
   - ダッシュボード → 更新
   - テーマの更新があれば適用

> ⚠️ 更新前には必ずバックアップを取ってください

### セキュリティスキャン

1. Wordfence → スキャン
2. 「新しいスキャンを開始」をクリック
3. 問題があれば対処

## トラブル時の対応

### よくある問題

| 症状 | 対処法 |
|------|--------|
| ログインできない | パスワードリセットを試す |
| ページが表示されない | キャッシュをクリア / サーバー状況確認 |
| 更新後に不具合 | バックアップから復元 |

### サポート連絡先

- Xserver サポート: support@xserver.ne.jp
- WordPress公式フォーラム: https://ja.wordpress.org/support/
