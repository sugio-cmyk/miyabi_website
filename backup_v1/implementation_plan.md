## 概要

要求定義書に基づき、Lightningテーマを使用したWordPressサイトを構築します。

## Phase 1: 初期構築

### 1.1 WordPress環境準備

| 作業 | 詳細 |
|------|------|
| WordPressインストール | Xserverの簡単インストール機能 |
| SSL設定 | Xserver無料SSL |
| テーマインストール | Lightning（公式から） |

### 1.2 必須プラグインインストール

| プラグイン | 用途 | 設定難易度 |
|------------|------|-----------|
| Lightning G3 Pro Unit | テーマ拡張（任意） | 低 |
| VK Blocks | ブロック拡張 | 低 |
| Contact Form 7 | 問い合わせフォーム | 低 |
| CF7 to Webhook | Slack連携 | 中 |
| UpdraftPlus | バックアップ | 低 |
| Site Kit by Google | GA4連携 | 中 |
| Wordfence | セキュリティ | 低 |

### 1.3 固定ページ作成

| ページ | スラッグ | 内容 |
|--------|----------|------|
| ホーム | home | ヒーロー、特徴、サービス導線（詳細はサービスページへ誘導） |
| サービス | services | **3サービス**: 社保手続き、給与計算、人事労務支援 |
| 事務所概要 | about | 代表挨拶、事務所情報 |
| お問い合わせ | contact | CF7フォーム、連絡先 |
| プライバシーポリシー | privacy-policy | テンプレート使用 |

### 1.4 メニュー・表示設定

- メインメニュー作成（4ページ）
- フロントページ設定（ホームページ＝home）
- パーマリンク設定（投稿名）

---

## Phase 2: 連携設定

### 2.1 Slack連携（CF7 to Webhook）

```
1. SlackでIncoming Webhook URLを発行
2. CF7 to WebhookプラグインにURL設定
3. 通知メッセージ形式を設定
4. テスト送信で確認
```

### 2.2 Google Analytics 4

```
1. Site Kit by Googleをインストール
2. Googleアカウント連携
3. GA4プロパティ作成/連携
4. 測定ID確認
```

### 2.3 バックアップ設定

```
1. UpdraftPlusでGoogle Drive連携
2. 週1回自動バックアップ設定
3. 保持世代数：4世代
```
