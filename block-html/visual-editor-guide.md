# ビジュアルエディタガイド - トップページ

このガイドでは、`home.html` をWordPressに貼り付けた後、
ビジュアルエディタで行う作業を説明します。

---

## 作業の流れ

1. `home.html` をコードエディタに貼り付け
2. 画像URLを置換（`[hero-bg.pngのURL]` など）
3. ビジュアルエディタに切り替え
4. **本ガイドに従い、プレースホルダーをVK Blocksに置換**

---

## 1. Trust Badges セクション（信頼バッジ）

### 場所
ヒーローセクションの直下、白背景のセクション

### 現在の状態
4つのカラムに `[ICON:xxx]` と表示されています

### 作業手順

各 `[ICON:xxx]` のパラグラフブロックを削除し、VK Blocks の**アイコン**ブロックに置換

| 置換対象 | Font Awesomeアイコン | サイズ | 色 |
|---------|---------------------|--------|-----|
| [ICON:user-tie] | `fas fa-user-tie` | 48px | #1a365d |
| [ICON:comments] | `fas fa-comments` | 48px | #1a365d |
| [ICON:laptop] | `fas fa-laptop` | 48px | #1a365d |
| [ICON:building] | `fas fa-building` | 48px | #1a365d |

### VK Blocks アイコンの設定方法

1. パラグラフ `[ICON:xxx]` を選択して削除
2. 「＋」から「アイコン」（VK Blocks）を追加
3. アイコン選択画面で Font Awesome アイコンを選択
4. サイドバーでサイズを `48px` に設定
5. 色を `#1a365d`（紺色）に設定

## 3. FAQセクション（よくある質問）

### 場所
「よくある質問」見出しの下

### 現在の状態
`[FAQ PLACEHOLDER]` と表示されたグループブロックがあります

### 作業手順

1. `[FAQ PLACEHOLDER]` を含むグループブロック全体を削除
2. VK Blocks の「**新FAQ**」ブロックを4つ追加
3. 以下の内容を入力

### FAQ内容

#### FAQ 1
- **質問**: 相談は無料ですか？
- **回答**: はい、初回のご相談は無料です。お気軽にお問い合わせください。ご相談内容をお伺いした上で、最適なサポート内容とお見積もりをご提案いたします。

#### FAQ 2
- **質問**: どのような企業が対象ですか？
- **回答**: 香川県内の中小企業様を中心にサポートしております。業種は問いません。社会保険・労働保険の手続きや給与計算でお困りの企業様はお気軽にご相談ください。

#### FAQ 3
- **質問**: オンラインでの相談は可能ですか？
- **回答**: はい、Zoomなどのビデオ通話を使ったオンライン相談に対応しております。ご来所いただく必要がないため、お忙しい経営者様にもご好評いただいております。

#### FAQ 4
- **質問**: 契約前に見積もりはもらえますか？
- **回答**: はい、ご契約前に必ずお見積もりをお出しします。ご依頼内容や従業員数などをヒアリングの上、明確な料金をご提示いたしますのでご安心ください。

### VK Blocks 新FAQの設定

1. 「＋」から「新FAQ」（VK Blocks）を追加
2. 質問欄に質問文を入力
3. 回答欄に回答文を入力
4. 必要に応じてスタイルを調整
   - アイコンデザイン: お好みで選択
   - 背景色: 白（#ffffff）推奨

## 5. サービスページ（services.html）のアイコン置換

### 場所
各サービスカード内の「こんな方におすすめ」「主な対応業務」見出し

### 現在の状態
絵文字（✓、≡）が使用されています

### 作業手順

| 現在の表示 | 推奨アイコン | サイズ | 色 |
|-----------|------------|--------|-----|
| ✓ こんな方におすすめ | `fas fa-check-circle` | 16px | #1a365d |
| ≡ 主な対応業務 | `fas fa-list` | 16px | #1a365d |

### 設定方法（上下ズレを防ぐ）

**方法1: グループブロックでFlex横並び**
1. 「＋」→「グループ」ブロックを追加
2. サイドバー → レイアウト → 「横一列」を選択
3. **垂直配置を「中央揃え」**に設定 ← これが重要！
4. グループ内に「アイコン」（VK Blocks）を追加
5. 同じグループ内に「見出し」または「段落」を追加
6. アイコン設定:
   - スタイル: **アイコンのみ**（noline）
   - サイズ: 16〜18px
   - 色: #1a365d

**方法2: インラインで使う場合**
見出しブロック内に直接アイコンを挿入すると、同じ行内に収まりますが
上下がずれやすいです。方法1の「グループ＋Flex＋中央揃え」が確実です。

### VK Blocksアイコンのスタイル選択

| スタイル | 見た目 | 用途 |
|---------|-------|------|
| アイコンのみ | シンプル | 見出し横、CTA連絡先 |
| 円 | ○ で囲む | Trust Badges、強調 |
| 四角 | □ で囲む | ボタン風 |

> **Note**: VK Blocksアイコンは見出しブロック内に直接挿入できない場合があります。
> その場合は、グループブロックでアイコンと見出しを横並び（Flex）にしてください。
